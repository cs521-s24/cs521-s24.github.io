--- layout: raw --- # Functions in C --- # Definition A block of code for one job Similar to Java's methods and Python's functions ```sh return_type name_of_the_function (parameters); ``` Commonly used methods: - int main(int argc, char* argv[]); - int printf(const char * restrict format, ...); --- # Why function? readability reuse of code testability --- # What becomes a function? - any repeated block of code - well-defined block of code with input and output - a block of code with many "internal" variables --- # How to design a function - the job description: what data does it modify? what actions? - the input and output: what types? which function calls this? - the test cases: test one thing at a time Example: [Huntington's Disease](https://www.genome.gov/Genetic-Disorders/Huntingtons-Disease) ---
